# HOW TO COMPILE THE PROGRAM 
************************************

To compile this program run 
$: gcc --std=gnu99 -o smallsh smallsh.c 

This will create an executable in the same directory called "smallsh" 

# TO RUN THE PROGRAM 
*************************************

While in the same directory as the executable file run 

$: ./smallsh
